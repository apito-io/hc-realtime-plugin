# hc-realtime-plugin

Real-time data synchronization via WebSocket for the Apito API platform.

## Features

- Real-time data sync
- WebSocket connections
- Conflict resolution
- Offline support
- Change tracking

## Installation

```bash
go build -o hc-realtime-plugin .
```
